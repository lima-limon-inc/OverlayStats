# ukui-overlay

Gentoo overlay for UKUI desktop environment

你可以添加这个仓库然后安装 ukui-meta 包来体验一个最小的 UKUI 桌面环境。

You can add this overlay and then emerge ukui-meta package to use a mininal UKUI desktop environment.
